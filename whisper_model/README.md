# Whisper Model

This directory is for the faster-whisper model files.

## Download

Run the download script before using the application:

```bash
python download_model.py
```

This will download the faster-whisper small model (~80MB) on first run.

## Manual Download

If you prefer, you can manually download the model by setting the `WHISPER_MODEL_PATH` environment variable to point to your local model directory.
